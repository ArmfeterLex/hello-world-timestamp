# Hello World Timestamp

Простейшее приложение на C# Windows Forms.

<img width="387" height="335" alt="Снимок экрана 2026-03-01 183841" src="https://github.com/user-attachments/assets/1f31c236-136e-442d-a647-448ae278d2b7" />

## Функциональность

- Одна кнопка на форме
- Один label для вывода
- При каждом нажатии в label добавляется "Hello World!" + текущее время

## Интерфейс

- button1 — кнопка для добавления записи
- label2 — вывод сообщений

## Куда можно встроить

- В демонстрационные приложения
- В логирование простых событий
- В учебные проекты по C#

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Для многострочного вывода label должен иметь достаточную высоту.
